# CRM Profesional con Google Sheets

Sistema completo de gestión de clientes (CRM) usando Google Sheets como base de datos, React para el frontend y Google Apps Script como backend.

## Contenido de esta carpeta

- **GUIA_YOUTUBE.md** - Guía completa paso a paso para construir el CRM desde cero
- **ClientesCRUD.gs** - Script de Google Apps para operaciones CRUD de clientes
- **AuthSystem.gs** - Script de Google Apps para autenticación y gestión de usuarios

## Repositorio del Proyecto

El código completo del proyecto está en:
https://github.com/SinCodigoLat/cliente-connect

## Características del CRM

- Autenticación de usuarios (Login/Logout)
- Gestión de clientes (Crear, Editar, Eliminar)
- Dashboard con métricas en tiempo real
- Pipeline de ventas (vista Kanban)
- Diseño responsive (móvil y desktop)
- Integración con Google Sheets (sin base de datos tradicional)
- Despliegue gratuito en Vercel

## Tecnologías

- React 18 + TypeScript
- Vite
- Tailwind CSS + Shadcn/ui
- Google Apps Script
- Google Sheets
- Vercel (hosting)
